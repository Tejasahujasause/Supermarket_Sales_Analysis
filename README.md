**Sales Analytics Dashboard (Power BI)**

This repository contains a complete Sales Analytics Dashboard built in Power BI. The project uses an Excel dataset with three tables and converts it into a structured, interactive report that highlights sales, profit, customer, product, and geographic insights.

**Files Included**
Sales data_FirstPBIX.xlsx – Full dataset with three tables
Sales_Analytics.pbix – Power BI dashboard file
Sales_Analytics_Dashboard.pdf – PDF version of the dashboard
README.md – Project overview

**Project Overview**
The goal of this project is to take raw sales data and build a clear, business-friendly dashboard. The report covers performance trends, profitability, customer behaviour, product insights, and region-wise metrics.

**What the Dashboard Shows**
Total sales, profit, order count, and return analysis
Month-wise sales and profit trends
Sales and profit by region, state, and city
Most profitable cities and operating regions
Product category and sub-category performance
Top selling products
Customer analytics including top customers and segment breakdown
Customer acquisition month-wise

**Power BI Features Used**
DAX measures for KPIs and calculations
Field parameters to switch between views dynamically
Data modeling following a clean relationship structure
Interactive visuals and slicers
Consistent formatting, layout design, and report structure

**Purpose of the Project**
This project was created to practice end-to-end Power BI development. It demonstrates the ability to clean data, model it properly, create DAX measures, build visuals, and design a complete dashboard that is easy to read and useful for decision making.
